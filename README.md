# UNES - Faculdade

Projeto desenvolvido utilizando HTML5 e CSS3 puro para uma faculdade fictícia chamada UNES.

## 📌 Sobre o projeto
Este projeto foi criado com o objetivo de praticar os fundamentos do desenvolvimento web front-end, utilizando apenas HTML e CSS, sem frameworks ou bibliotecas externas.

O site possui uma interface simples e organizada, contendo páginas institucionais de uma faculdade fictícia.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3

## 📂 Páginas do site
- Página Inicial
- Quem Somos
- Contato
